# Bookshelfie

Bookshelfie is a [React](https://github.com/facebook/create-react-app) application where users can track their library of read and unread books, as well as write and update reviews for them.

## Viewing the app

You can view a demo version of the Bookshelfie app [here](https://bookshelfie.netlify.app/).

Or you can view the app locally by forking and cloning the app, before using the following commands:

1. `npm install`
2. `npm start`
